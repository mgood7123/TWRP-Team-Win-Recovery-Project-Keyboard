# TWRP-Team-Win-Recovery-Project-Keyboard
a recreation of the Team Win Recovery Project keyboard as an IME


## contains
* IME Keyboard x1

## features
* Control Key
* Alt Key (violates official TWRP keyboard look)
* Escape Key
* Tab Key
* Arrow keys
* Alphabetical keys
* symbols key
* `/` key
* space key
* `.` key
* return key
* shift key
* backspace key
